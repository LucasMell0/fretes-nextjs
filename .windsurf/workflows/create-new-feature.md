# Workflow: Create New Feature

1. Analyze if similar feature already exists.
2. Identify reusable components.
3. Check if shadcn components can solve it.
4. Ensure mobile-first responsiveness.
5. Apply tenant isolation if needed.
6. Follow existing folder structure.
7. Keep code clean and typed.
8. Avoid duplication.
